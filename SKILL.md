## 理论渊源

### 《证病结合用药式》核心思想

本书是欧阳锜先生毕生学术经验总结，创造性提出"**三型二十一证互为纲目**"的辨证体系：

1. **认知发展**：中医认识疾病的过程是 `症 → 病 → 证`
   - **症**：疾病外在表象（症状、体征）
   - **病**：疾病全过程，由特殊本质决定
   - **证**：疾病某一阶段的本质反映，包含病因、病位、病性、病势四要素
   - **核心**：证是连接"病"与"治"的关键环节，是"异病同治"和"同病异治"的理论基础

2. **辨证求衡理论**：中医诊疗的核心思维是**平衡理论**
   - 治疗目标：恢复人体内环境动态平衡
   - **求衡四法**：
     - **正面求衡**：直接针对明显失衡，如"寒者热之、虚者补之"
     - **反面求衡**：透过现象看本质，用于真假寒热复杂证
     - **按比例求衡**：寒热/虚实夹杂，按比例决定用药主次

3. **主症定量**：辨证的关键在于抓住**主症**
   - 主症：反映疾病本质、起决定性作用的症状
   - 主症变化不仅是量的增减，更揭示证与证之间**质变**的传变规律

4. **三型二十一证体系**：将所有证候归纳为三个基本类型：

| 型 | 类别 | 二十一纲领证 | 说明 |
|----|------|--------------|------|
| **第一型** | 五气为病 | 风、热（暑、火）、湿、燥、寒 | 病因：外感/内生邪气 |
| **第二型** | 脏腑主病 | 肝、心、脾、肺、肾、胆、小肠、胃、大肠、膀胱 | 病位：内在脏腑失调 |
| **第三型** | 邪留发病 | 痰、饮、水气、瘀血、食积、虫积 | 病理产物：邪结停留 |

### "互为纲目"的精髓

三个维度地位平等，如同立体坐标系的三个轴（X, Y, Z）：

- 任何具体复杂证候，都是三个轴向上不同证的**组合投影**
- 分析时，某一型可以是"**纲**"（主要矛盾），其他型则是"**目**"（次要矛盾），反之亦然
- 天然贯通**外感与内伤**，打破"外感用伤寒温病，内伤用脏腑"的思维定势

### 辨证五步法

```
步骤一：信息采集     → 收集症状、舌象、脉象
步骤二：三维归因     → 在五气、脏腑、邪留三个维度分别匹配计分
步骤三：分析关系     → 判断因果链、主次先后、真假同异
步骤四：确定证型     → 明确纲-目结构，得出辨证结论
步骤五：指导治疗     → 给出治疗策略和推荐方剂
```

## 使用方式

### 触发方式

你只需要说出：`三维辨证：` 加上你的病例症状，例如：

```
三维辨证：患者发热恶寒，头痛，无汗，苔薄白，脉浮紧
```

或者：

```
请用三维辨证分析这个病例：年龄42岁, 性别女，月经量少，八天干净，色暗有血块，怕冷，腰酸疼，双尺脉不及，眼睛干涩，睡眠不好
```

## 完整体系结构

### 三型二十一证纲领

**五气为病**（6证）：
- 风、热、暑、湿、燥、火（暑与火同类）

**脏腑主病**（10证）：
- 肝、心、脾、肺、肾、胆、小肠、胃、大肠、膀胱

**邪留发病**（5证）：
- 痰、饮、水气、瘀血、食积、虫记

合计：**21** 个纲领证 → "三型二十一证"

---

## 辨证关键

在复杂证候中，需通过三大关键抓准主症：

1. **辨轻重缓急** → 确定主次
2. **辨先后因果** → 确定原发继发
3. **辨真假同异** → 识破寒热虚实真假

掌握传变：通过"**六变**"（表里出入、寒热转化、虚实消长）预判疾病发展趋势。

## 示例

### 病例：月经量少，怕冷，腰酸疼，色暗有血块

**三维归因结果**：

| 维度 | 结果 | 置信度 |
|------|------|--------|
| 五气 | 寒 | 95% |
| 脏腑 | 肾 | 85% |
| 邪留 | 瘀 | 90% |

**辨证结论**：
- 纲：邪留-瘀
- 目：五气-寒，脏腑-肾
- **证型**：肾虚寒凝，瘀血内阻

**治疗策略**：
- 主治：活血化瘀（祛邪留）
- 兼治：温补肾阳（调脏腑），温散寒邪（祛五气）
