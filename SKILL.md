# 中医三维辨证辅助诊断

基于**欧阳锜《证病结合用药式》**"五气-脏腑-邪留"**三维辨证框架**的临床决策支持系统。

## 核心理念

### 三维辨证体系

将任何复杂证候，都分解为**三个维度**上的组合，并分析其主次、因果、转化关系：

```
┌─────────────────────────────────────────────┐
│           三维辨证体系                        │
├─────────────────────────────────────────────┤
│  🔴 五气维度（外感/内生邪气）                  │
│     风、寒、暑、湿、燥、火                     │
│     → 追溯病因起源                            │
├─────────────────────────────────────────────┤
│  🟠 脏腑维度（病位定位）                       │
│     肝、心、脾、肺、肾、胃、胆等               │
│     → 定位核心病位                            │
├─────────────────────────────────────────────┤
│  🟢 邪留维度（病理产物）                       │
│     痰、饮、水、湿、瘀、食积                   │
│     → 分析病理产物                            │
└─────────────────────────────────────────────┘
```

### 纲-目结构

用"纲-目"结构命名证候，明确主次：

> **纲**：当前主要矛盾（急则治标）
> **目**：次要/根本问题（缓则治本）

示例：
- 以「五气-寒」为纲，以「脏腑-肺」为目
- 以「邪留-痰」为纲，以「脏腑-肺」为目

## 理论来源

本技能基于国医大师**欧阳锜**先生所著《证病结合用药式》的三型辨证体系：

> "三纲鼎足、互为纲目的辨证体系，以五气、脏腑、邪留三个类型为纲，各类证候为目，形成互为纲目的结构模式。"

## 适用人群

- **基层医生**：提高辨证的规范性和准确性
- **中医学习者**：理解辨证思路和证候鉴别

## 核心框架

### 三维辨证体系（101证）
│  C. 邪留发病（病理产物）                      │
│  ├── 瘀血阻络、瘀阻脑络                      │
│  ├── 痰浊上逆、痰热互结                      │
│  ├── 水溢皮肤、饮食积滞                      │
│  └── 共6证                                  │
└─────────────────────────────────────────────┘
```

## 使用方法

### 诊断流程

```
用户输入症状描述
        │
        ▼
系统解析症状 → 症状标准化映射
        │
        ▼
三维归因分析:
  • 五气维度评分（风、寒、暑、湿、燥、火）
  • 脏腑维度评分（肝、心、脾、肺、肾）
  • 邪留维度评分（痰、饮、水、瘀、食积）
        │
        ▼
分析因果关系 → 判断主次、因果链
        │
        ▼
确定证型 → 纲-目结构命名
        │
        ▼
输出诊断报告:
  • 三维维度分析
  • 证候诊断结果
  • 方药建议（含组成、加减、注意事项）
  • 鉴别要点
```

### 命令行使用

```bash
# 交互式诊断
python3 scripts/diagnose.py

# 直接输入症状
python3 scripts/diagnose.py --symptoms "发热,恶寒,无汗,头身疼痛" --pulse "浮紧" --tongue "苔薄白"

# 简化输入
python3 scripts/diagnose.py 发热 恶寒 头痛 无汗 咳嗽 痰白
```

## 注意事项

1. **辅助工具性质**：本系统为辅助诊断工具，不能替代医生临床判断
2. **信息完整性**：症状描述越完整，诊断越准确
3. **脉诊舌诊**：舌脉信息对证候判断至关重要
4. **个体差异**：同一证候在不同患者可能表现不同
5. **动态观察**：病情变化时需重新辨证
6. **证病结合**：建议结合具体疾病进行辨证，因病而异

## 算法说明

### 置信度计算

```
置信度 = 主症匹配度×0.5 + 次症匹配度×0.25 + 舌象匹配度×0.15 + 脉象匹配度×0.1

最终置信度 = 症状置信度×0.7 + 维度匹配度×0.3
```

### 维度评分

- **五气维度**：追溯病因起源，识别风、寒、暑、湿、燥、火的影响程度
- **脏腑维度**：定位核心病位，识别肝、心、脾、肺、肾等功能失调
- **邪留维度**：分析病理产物，识别痰、饮、水、瘀、食积等

## 辨证流程（五步法）

```
步骤一：信息采集 → 收集症状、体征
步骤二：三维归因 → 将症状归入五气、脏腑、邪留三个维度
步骤三：分析关系 → 判断因果链、主次关系
步骤四：确定证型 → 用纲-目结构命名证候
步骤五：指导治疗 → 治法方药对应纲-目结构
```
