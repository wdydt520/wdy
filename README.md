# 中医三维辨证辅助诊断系统

基于**欧阳锜《证病结合用药式》**「五气-脏腑-邪留」三维辨证框架的临床决策支持系统。

## 核心特点

- **三维辨证框架**：五气（病因）+ 脏腑（病位）+ 邪留（病理产物）
- **纲-目结构**：明确主次，指导治疗
- **101个证候**：五气为病41证 + 脏腑主病45证 + 邪留发病15证
- **133首方剂**：含组成、功效、加减、注意事项
- **4090条疾病应用**：各疾病辨证要点与方药加减

## 辨证流程（五步法）

```
步骤一：信息采集 → 收集症状、体征
步骤二：三维归因 → 五气 + 脏腑 + 邪留
步骤三：分析关系 → 判断因果链、主次关系
步骤四：确定证型 → 纲-目结构命名
步骤五：指导治疗 → 方药建议
```

## 快速开始

### 网页版使用

用浏览器打开 `web/index.html` 即可使用

### 命令行使用

```bash
# 安装依赖
pip install pandas

# 运行诊断
python scripts/diagnose.py --symptoms "发热,恶寒,无汗,头痛" --tongue "苔薄白" --pulse "浮紧"
```

## 理论来源

本技能基于国医大师**欧阳锜**先生所著《证病结合用药式》的三型辨证体系：

> "三纲鼎足、互为纲目的辨证体系，以五气、脏腑、邪留三个类型为纲，各类证候为目，形成互为纲目的结构模式。"

## 注意事项

1. 本系统为辅助诊断工具，不能替代医生临床判断
2. 症状描述越完整，诊断越准确
3. 舌脉信息对证候判断至关重要
